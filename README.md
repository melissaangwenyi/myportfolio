# Developer Portfolio

Personal portfolio website built with HTML, Tailwind CSS, and JavaScript. Showcases projects, technical skills, and contact information.

## Tech Stack

- HTML5
- Tailwind CSS (CDN)
- JavaScript
- Font Awesome icons
- Google Fonts (Inter, JetBrains Mono)

## Setup

```bash
git clone https://github.com/melissaangwenyi/myportfolio.git
cd myportfolio
```

Open `index.html` in a browser. No build step required.
